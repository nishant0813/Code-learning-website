Introduction to HTML: Understanding the Language of the Web

what is HTML?

  • HTML, or HyperText Markup Language,is the Languageof the web, used to creat websites.
  • It defines the barebone page structure or layout of web pages, also known as webpages, that we see on internet .
  • HTML consist of a set of tags written in an HTML Document, and files typically have a ".HTML" or ".HTM" extension.
  • There are several versions of HTML, with HTML5 being the latest.

Features of HTML

  • It is platform-independent.
  • Images, videos, and audio can be added to a web page.
  • Hypertext can be added to the text.
  • It is a markup language.
  • It can be integrated with other languages like CSS, JavaScript,, etc.
  • Semantic Structure.

Why the Term HyperText & Markup Language?

The word hypertext markup language comprises the words “hypertext” and “markup language”. The term "hypertext" refers to the linking of text with other documents and “markup language” refers to a language that uses a set of tags. So, HTML is the linking of text with other documents using some set of tags. Note:- Tags refer to some meaningful texts enclosed in angle braces <___>. For eg. <_anytag_>. Each tag has a different meaning and importance in building an HTML page which can affect the web page in its own ways.

A beautiful analogy to understand HTML:

In building a webpage, you can think of HTML, CSS, and JavaScript as different parts of a car. Imagine that HTML is the car's skeleton, forming the basic structure and frame. Next, CSS is like the paint and finishing touches that make the car look appealing, adding color, style, and design. Finally, JavaScript is the engine and other mechanical parts, bringing the car to life by adding functionality, movement, and interactive features. In the same way, when developing a website, HTML lays out the structure, CSS makes it visually appealing, and JavaScript adds interactivity and dynamic content.

History of HTML:

  • In 1989, Tim Berners Lee established the World Wide Web(www) and created HTML in 1991.
  • From 1995 to 1997, he started working on versions of HTML.
  • In 1999, a committee was organized that madeHTML4.0 a standard.
  • HTML4.0 is used by many, even today. However, the stable version of HTML is 5.0 also known as HTML5.
